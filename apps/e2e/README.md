# E2E Tests (Playwright)

Shared Playwright project for end-to-end tests across frontend and backend.

## Run tests

```bash
npm test
```

## Run interactive UI mode

```bash
npm run test:ui
```

## Base URLs

- `FRONTEND_URL` (default: `http://localhost:4200`)
- `BACKEND_URL` (default: `http://localhost:3000`)
